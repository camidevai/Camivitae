### User Story
As a user, I want to drag and drop my resume file so that I can easily upload it.

### Acceptance Criteria (Gherkin)
```gherkin
Scenario: Upload File
  Given I am on the dashboard
  When I drag a PDF file into the dropzone
  Then the UI should show the file name
  And the "Analyze" button should become active
```
