### User Story
As a user, I want to download the formatted resume so that I can apply to jobs.

### Acceptance Criteria (Gherkin)
```gherkin
Scenario: Download PDF
  Given I am viewing the preview
  When I click "Download"
  Then a PDF file should be downloaded
  And the PDF should be selectable text (not image)
```
