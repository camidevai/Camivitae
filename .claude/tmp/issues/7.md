### User Story
As a frontend, I want to send a file to an API endpoint so that I can get the parsed text back.

### Acceptance Criteria (Gherkin)
```gherkin
Scenario: Call API
  Given I have a file
  When I POST to "/api/parse"
  Then I should receive a JSON response with the text
```
